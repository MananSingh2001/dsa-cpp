# DSA in C++ (One Problem, One Commit 🎯)

My comprehensive repository for Data Structures and Algorithms practice. Each commit represents exactly one solved problem, ensuring consistent progress and a detailed history of my learning journey.

## 📈 Learning Habit
- **Consistency is key**: Five problems, five commits per week minimum.
- **Micro-commits**: Each solution is pushed as a standalone commit with a clear, descriptive message.
- **Green Square Strategy**: Sustained daily activity demonstrating mastery and dedication.

## 📂 Structure
- `arrays/`: Vectors, sliding windows, hashing.
- `strings/`: Pattern matching, manipulation.
- `linked-lists/`: Pointers and node management.
- `trees/`: Binary Search Trees, DFS, BFS traversal.
- `graphs/`: Topo sort, Dijkstra, BFS/DFS cycles.
- `dynamic-programming/`: Memoization and tabulation.
